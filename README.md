# 🚀 Personal Dashboard

🌐 **[Canlı Demo](https://dashboard-8rk5.onrender.com/)**

Kişisel kullanım için geliştirilmiş, tek sayfadan oluşan bir **fullstack dashboard** uygulaması. Günlük hayatı kolaylaştırmak için birçok özelliği tek ekranda sunar.

---

## ✨ Özellikler

| Özellik | Açıklama |
|---|---|
| 🕐 **Saat & Takvim** | Gerçek zamanlı saat, aylık takvim, takvimde ki günlere not ekleme ve 2 farklı şehrin saatini aynı anda görme (büyük/küçük format) |
| 🌤️ **Hava Durumu** | İstediğin şehri yazarak güncel hava durumu bilgisi alma |
| 🔗 **Kısayollar** | Sık kullanılan sitelere hızlı erişim linkleri |
| 🔔 **Hatırlatıcılar** | Zamanlı ve tekrarlayan hatırlatıcı oluşturma |
| ✅ **Yapılacaklar** | Görev ekleme, tamamlama ve takip |
| 📝 **Hızlı Notlar** | Anlık not alma alanı |
| 🍅 **Pomodoro** | Çalışma/mola zamanlayıcısı, günlük ve haftalık kişisel istatistikler |
| 📈 **Piyasalar** | Döviz, kripto ve altın fiyatları |
| 🎵 **Spotify** | Spotify entegrasyonu ile müzik kontrolü |
| 👋 **Kişiselleştirilmiş Karşılama** | Kullanıcı adına ve günün saatine göre dinamik karşılama mesajı (günaydın / iyi günler / iyi akşamlar / iyi geceler) |
| 🔐 **Hesap Sistemi** | Kayıt ve giriş sistemi — her kullanıcı kendi verilerini (notlar, görevler, hatırlatıcılar, kısayollar vb.) ayrı ayrı görür |
